A full-stack skill bank app done as a group project.

## Technologies used

### Frontend

- Figma for the UI design
- Cypress for e2e testing
- React.js
- Material UI

### Backend

- Express.js
- PostgreSQL
- Prisma ORM

### Devops

- Gitlab
- Docker
- Azure
